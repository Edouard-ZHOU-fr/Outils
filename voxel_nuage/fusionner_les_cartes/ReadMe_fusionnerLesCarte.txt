
Comment utiliser?

assurer tu as bien installée open3d:
pip install open3d

suivie les consignes dans le terminal.

1. Saisir le repertoire origine (avec/ à la fin)
2. Saisir le répertoire aussi le nom que tu veux sauvgarder (avec '.pcd' à la fin)

ex:

/media/hongyu/Setup4To/UTAC/Carton_UTAC/
/media/hongyu/Setup4To/UTAC/Carton_UTAC.pcd

Output sera dans la repertoire /media/hongyu/Setup4To/UTAC/      qui s'appelle Carton_UTAC.pcd
